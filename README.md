# Petra

Clipboard manager for Linux.

**Version 1.0.0**

## Features

- Keyboard navigation
- Filter grouping (text, images, links, commands)
- Emoji insertion
- Image preview
- Linux command recognition
- Pin items
- Pin window
- Clear items

![Screenshot 1](screenshots/screenshot1.png)

## Installation

In a terminal, run these commands in the project folder:
```bash
./build-flatpak.sh
./make-flatpak.sh
./install-flatpak.sh
```
